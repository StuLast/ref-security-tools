# Extenshun

##  Overview

This node.js application is designed to search installed VSCode extensions and compare against a list of known VSCode extensions that have been compromised by hackers.

##  To use

1.  Make sure you have node.js installed
2.  Run `node extenshun.js` from this folder.

The default location of the extensions folder is assumed to be the default location:

- Windows: %USERPROFILE%\.vscode\extensions
- macOS: ~/.vscode/extensions
- Linux: ~/.vscode/extensions
